Programming for the Web

Programming for the Web, or PWEB, is a Duke TIP Summer Studies course on programming using programs including HTML5, CSS3, and more. In this class, you will learn how to build a fully functioning website.

HTML5

HTML5 is the first program we learned. It lets us make a basic website.

CSS3

CSS3 is a program we use that allows us to style a website. It can do things such as changing colors and resizing/moving elements on the page.

Javascript

Javascript is a programming language that allows us to do many interactive elements. We currently have not learned Javascript, but we will soon and it will make our websites out of this world!

Bootstrap 4

Bootstrap 4 is a program which adds many CSS classes for easy use. It allows complex things like spinners and carousel sliders to be added.
